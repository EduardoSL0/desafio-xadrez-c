# ♟️ Desafio: Movimentando as Peças do Xadrez

&gt; **Nível:** Novato  
&gt; **Linguagem:** C  
&gt; **Tema:** Estruturas de Repetição (for, while, do-while)

---

## 📋 Descrição do Projeto

Este projeto simula o movimento de três peças de xadrez utilizando diferentes estruturas de repetição em linguagem C. O objetivo é demonstrar a compreensão e aplicação prática dos loops `for`, `while` e `do-while`.

---

## 🎯 Funcionalidades

| Peça | Movimento Simulado | Estrutura Utilizada | Casas |
|------|-------------------|---------------------|-------|
| **Torre** | 5 casas para a **Direita** | `for` | 5 |
| **Bispo** | 5 casas na **Diagonal** (Cima + Direita) | `while` | 5 |
| **Rainha** | 8 casas para a **Esquerda** | `do-while` | 8 |

---
